Internationalization (i18n)

It is a process of developing products in such a way that they can be localized for languages easily.



Document Type Definition (DTD)

It defines the legal blocks of an XML Documents. It is a set of markup declarations that define a document type for an SGML-family markup language (SGML, XML, HTML).



EXTENSIBLE HYPERTEXT MARKUP LANGUAGE (XHTML)

  • Reformulation of HTML
  • Much stricter compared to HTML.
  • Hybrid language between XML and HTML.
  • XHTML can be validated using XML tools.
  • It provides more cleaner and well structure format which helps in making the web pages easily.
  • It can support wide range of applications
