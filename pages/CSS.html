What is CSS?

  • CSS or Cascading Style Sheets is used to display HTML elements from one or more web pages.
  • Instead of creating documents, it allows HTML to have various styles such as:
    • Color
    • Design
    • Layouts
    • Fonts
    • Screen Size(depending on the devices)

CSS Versions

  • CSS1(December 1996).
  • CSS2(May 1998).
  • CSS3(June 1999).
  • CSS4(March 2017).

Who created CSS?

  • CSS was created by a group called the CSS Working Group, who worked within the World Wide Web Consortium or the W3C.
  • They define these documents as specifications. As the specifications are approved by the W3C members, it became recommendations, because the W3C have no control over the implementations pf the language.

Why use CSS?

  • It is easy to work on during maintenance.
  • Gives web pages a unique look.
  • Style sheets are saved via .css file.

How to use CSS?

  • When starting a browser, the stylesheets formats the .css file.
  • There are three ways of inserting it:

    • External Stylesheet
      changes the appearance of the web page entirely.

    • Internal Stylesheet
      is applied on one page but with a different style.

    • Inline Styles
      uses a single element within the unique style.
